퀴즈 데이터 생성기

텍스트 형식으로 문제를 입력하여 퀴즈 데이터 파일을 생성하고 다운로드하세요.

파일 정보

입력 형식 안내

퀴즈 데이터 입력 형식

각 문제는 빈 줄로 구분하며, 첫 번째 줄은 문제, 나머지는 선택지입니다. 정답은 *로 표시하세요.

문제 1: JavaScript에서 변수를 선언하는 키워드는?
*let
const
var
function

문제 2: HTML의 기본 구조에서 제목을 나타내는 태그는?
div
span
*h1
p

문제 3: CSS에서 글자 색상을 지정하는 속성은?
*color
font-size
margin
padding

참고: 생성된 퀴즈 데이터는 모든 퀴즈 유형(고정형, 랜덤형, 순서 변경형, 암기 카드형)에서 사용할 수 있습니다.
